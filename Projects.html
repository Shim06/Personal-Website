PandesalEmu

PandesalEmu is an emulator for the Pandesal CPU, an 8-bit processor inspired by the MOS 6502. This is a CLI-based program that will execute Pandesal binary programs.

PandesalAssembler

PandesalAssembler is an assembler for the Pandesal CPU, an 8-bit processor inspired by the MOS 6502. This is a CLI-based program that reads asm source files and outputs a .bin binary ready to be executed.

PandesalCPU

Pandesal CPU is an 8-bit multi-cycle general-purpose CPU architecture inspired by the classic MOS 6502. Built at the logic-gate level and implemented in Logisim Evo.

Nibbelium

Nibbelium is a CHIP-8 Emulator written in C++ with Dear ImGui and SDL2 for Windows. It has been tested with over 100 game ROMs and 8 test ROMs to ensure compatability and accuracy in emulation.

Terraria Fishing Bot

Terraria Fishing Bot is a program that uses image recognition to fish automatically in Terraria. It uses the PyAutoGUI and OpenCV libraries to scan the screen and check for any detections. The program uses CustomTkinter for its front-end UI. It includes features such as remembering the last catch list used and auto drinking potions in the game. A statistics section is also included to see how much catches have been caught using the program.
